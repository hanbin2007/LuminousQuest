# LuminousQuest 视觉素材风格规范

制定与终审:Fable 5 | 版本:v1(2026-07-15)
适用:全部拖拽组件位图、案例装置简图与结构剖面图。素材由 codex 生成,**每批产出须对照本规范由 Fable 5 签核**。

## 1. 总原则

- **教辅彩色插图风,本质是抽象后的教学简图**:明快、干净、示意性;教学指向永远优先于装饰。像苏教版教材插图的现代化版本,不是儿童卡通,也不是写实渲染。
- 素材将呈现在**纸白界面(#F7F8F5)**上:透明底,不自带背景;配色要在纸白上"坐得住"。
- **位图内禁止出现任何文字**(含化学式、正负号标注)。全部标注由 UI 层矢量渲染——保证内容可配置、字体统一、离线不缺字。方向箭头同理由 UI 层 SVG 绘制(需旋转与语义着色),**不出位图**。

## 2. 色板(与设计 token 同源,禁自创颜色)

| 角色 | 色值 | 用途 |
|---|---|---|
| 墨线 | `#1F262B` | 全部描边、深色细节 |
| 校深蓝 | `#2B5CA8` | 结构件主体(容器沿口、支架、导线) |
| 校青 | `#2CB8B8` | 溶液、水系、离子相关(液体填充用 20–35% 不透明度) |
| 琥珀 | `#E8960C` | 能量与高亮(电流表指针、灯丝发光) |
| 铜橙 | `#C4703A` / 锌灰 `#9AA3AB` / 石墨深灰 `#3A4148` | 真实材质点缀(铜片、锌片、碳棒) |
| 浅色系 | 天青浅 `#DFF3F2`、蓝浅 `#DEE9F9`、琥珀浅 `#FBEBD2` | 填充与高光 |

## 3. 造型语言

- 圆角与胶囊造型为主;转角圆润但不软塌。
- **统一描边**:墨线色,视觉粗细约为部件宽度的 2.5%(256px 部件 ≈ 6px@2x),全套素材必须一致——这是批次一致性的第一检查项。
- 平涂+单层浅色高光;**无投影、无渐变网格、无外发光**(发光效果由 UI 层做)。
- 液体:校青半透明平涂 + 一条水平液面高光;气泡为空心小圆。

## 4. 部件规格(拖拽组件)

- 透明底 PNG,**2x 输出**:基准画布 512×512(逻辑 256),部件居中、四周留白 ≥8%。
- 首批试产清单(过审后才排批量):①惰性碳棒电极 ②金属片电极(锌灰) ③烧杯+校青溶液 ④盐桥(U 型管) ⑤电流表。
- 全量清单由 `assets/manifest.json` 登记:`assets/components/<id>@2x.png`;案例图 `assets/cases/<caseId>/<view>.png`(横版 4:3,3200×2400@2x)。剖面图允许局部材质纹理,占比 ≤20% 画面。

## 6. 批次验收 Checklist(Fable 5 签核,逐项打勾)

1. 透明底干净:alpha 边缘无白边、无杂点(放大 400% 抽查)。
2. 色板符合:主色取样与规范 hex 目视无偏差。
3. 描边粗细与既往批次一致(与 anchor 图并排对比)。
4. 造型语言一致:圆角程度、高光方式同族。
5. 无文字、无箭头、无阴影。
6. 机检通过:尺寸/命名/透明通道/manifest 登记(脚本自动)。
7. 与纸白底(#F7F8F5)合成预览无违和。

任一项不过 → 整批退回重生成;连续两批不过 → 触发退路(Fable 5 亲自出图或缩减素材规模)。

## 7. 真实比例规范(v1.2,2026-07-17,工作台重构)

**位图必须携带现实比例**:比例语义由 `src/features/builder/presentation.ts` 的 `benchGeometryFor` 显示几何表承载,位图纵横比必须与显示几何完全一致(见 §8),UI 等比渲染不得拉伸重采样。

## 8. 工作台半写实规格(v1.3,2026-07-17,素材质量重画)

**适用范围:仅工作台 8 张精灵位图**(其余素材继续走 §1–§6 教辅简笔风)。工作台走 NOBOOK 式虚拟实验质感——**半写实渲染精灵**,不是简笔画:器材要一眼认出是"真的实验器材",但仍是干净的矢量渲染,不是照片。v1.3 与 §3 冲突之处以本节为准。

### 8.1 画布与分辨率(机检硬约束)

画布 = 显示几何 × **3**(覆盖 dpr2 + 1.5× 头寸,杜绝放大取糊),纵横比与 `benchGeometryFor` **逐像素一致**:

| 资产 | 显示几何 | 画布(px) |
|---|---|---|
| electrode-carbon / electrode-zinc / copper | 36×168 | **108×504** |
| wire / insulated-wire | 150×70 | **450×210** |
| beaker-empty / beaker-electrolyte / sucrose-beaker | 210×260 | **630×780** |

### 8.2 几何契约(判分/浸没渲染依赖,容差 ±2%)

装配推导(`assembly.ts`)与浸没覆绘(`bench-renderer.ts`)按包围盒比例取区域,位图必须吻合:

- **电极**:主体竖直板/棒,占画布宽 ≥85%;顶端接线端在 y≈2%–8%(鳄鱼夹由渲染器咬在 anchorY=0.05 处);底端边缘落在 y≈94%–98%(`bottomOf` 取 0.96)。上下左右留白合计 ≤6%。
- **烧杯**:玻璃**内壁**位于 x=12% 与 x=88%(±2%);**液面线在 y=32%**(±2%);液体/杯内底到 y≈96%–98%。杯口沿、尖嘴、外壁允许略超出这些线,但液体像素必须严格在此矩形内——浸没效果就是把这张图按该矩形裁剪后覆绘在电极上,错位即穿帮。
- **液体必须半透明**(见 8.3),浸入的电极要能透出来。

### 8.3 渲染语言(替代 §3,半写实)

- **禁止粗墨描边**。轮廓由明暗关系呈现;允许 ≤1.5px、不透明度 ≤30% 的深色缘线收边。
- **金属(电极/夹子)**:纵向多段线性渐变(≥5 个停靠点),左缘窄高光带、右缘暗带、顶端截面用小椭圆亮面收口,底缘压暗。
  - 锌:冷灰蓝 `#C6CDD5→#7E8992` 族(以 `#9AA3AB` 为中间调);
  - 铜:暖铜 `#E29A5C→#8C4E22`(以 `#C4703A` 为中间调),加一条竖直镜面反光条;
  - 石墨:哑光近黑 `#4A5158→#23282E`(以 `#3A4148` 为中间调),叠 2–3 条极低对比纵向纹理条,亚光、几乎无镜面。
- **玻璃(烧杯)**:杯体白色 8%–16% alpha 填充;左侧一条纵向高光带(白 35%–55% alpha,用 2–3 层退阶矩形做柔边);杯口椭圆沿口亮暗双线;尖嘴;杯壁厚度用内外双线暗示;杯底 6% 高度加厚微亮;刻度为蚀刻感短横线(白 ≤25% alpha),**无数字**。
- **液体**:
  - electrolyte:校青纵向渐变 `#5BC0BC→#2E9E9A`,55%–70% alpha,液面画俯视椭圆 + 1.5px 高光弧;
  - sucrose:近无色(白 ≤8% alpha,极浅液面线),杯底一小堆白色半透明糖粒(不规则小多边形),**不得有任何"不导电"暗示标记**;
  - beaker-empty:同款玻璃,无液体。
- **导线**:沿 S 曲线的三层描边模拟圆管(暗缘/主体/高光芯线)。wire 校深蓝 `#2B5CA8` 光泽胶皮;insulated-wire 深灰哑光 `#3A3F45` 胶皮 + 夹子戴绝缘护套,两者一眼可辨。鳄鱼夹:镀镍金属渐变、可见锯齿咬口、铜色压接筒。
- 沿用:透明底、**无文字**、无方向箭头(UI 层画)、无环境投影。

### 8.4 工艺约束(光栅化器限制)

- 管线固定:`scripts/generate-real-scale-components.mjs`(SVG 生成)→ `scripts/rasterize-svg.swift`(NSImage 光栅化)→ `scripts/verify-real-scale-components.mjs`(机检)→ `assets/manifest.json` 登记。
- **禁用一切 SVG `<filter>`**(feGaussianBlur 等,NSImage 支持不可靠):柔边、光晕一律用多层透明度退阶的形状叠加实现;渐变只用 `linearGradient` / `radialGradient`。
- 渐变、透明度叠加以最终 PNG 为准逐张目检——SVG 里好看不算数。

### 8.5 v1.3 验收 Checklist(Fable 5 签核)

1. 8 张尺寸与 8.1 表逐像素一致;透明底、边缘无白边(机检+400% 抽查)。
2. 几何契约:烧杯液面 32%、内壁 12%/88%,电极端点 5%/96%(叠加参考线核对)。
3. 质感达标:金属有体积感、玻璃透、液体透;**任何一张仍像简笔画即整批退回**。
4. wire 与 insulated-wire 缩到 64px 托盘缩略图仍一眼可辨。
5. 在画布深灰底(#42474d)与纸白托盘(#F7F8F5)上双底色合成预览均不违和。
6. 浸没实测:电极插入 electrolyte 池,液下部分透色正确、裁剪线与液面吻合。
