# 直接评判实验报告（M6 v2 §3 准入证据）

日期：2026-07-23 ｜ 执行：Fable 5 ｜ 目的：裁决「agent 自主评判可否作为权威判分」

## 设置

52 个人工标注用例（26 学生 × D3/P1），direct-judge 模式：把题目、节点定义、
**题目专属参考答案要点**（question.evidence.referenceAnswerPoints）、误区目录全部
给模型，由其直接输出 hit/partial/miss + 误区 id。3 руны/case（多数票），5 路并发，
provider = claude-agent × claude-sonnet-5（temperature 0.1，无重试，呈现裸性能）。
对照组：同语料上的抽取+规则管线（第二轮校准，94.23%）。

注：首次尝试仅给通用节点量表（无题目答案要点）时判分完全失锚（首例即误判），
已修正——**直接评判可行的前提之一就是为每道题维护题目级答案要点**，这本身即
内容维护负担。

## 结果

| 指标 | 直接评判 | 抽取+规则管线 | 结论 |
|---|---:|---:|---|
| 宏命中率（多数票） | **57.69%** | 94.23% | 相差 36.5pt |
| 全错判掌握 | **1/13**（s29-p1 miss→hit） | 0/13（两轮均零） | 直接评判触发最危险违规 |
| 答错被多给分 | 6 case（5 miss→partial + 1 miss→hit） | 0 | 宽判方向系统性失守 |
| 三跑一致率 | 50.00% | 88.46% | 裁量自由度带来强波动 |
| 调用失败（无重试） | 30/156 | —（服务层有重试） | claude-agent 裸噪声 |

错判模式：13× hit→partial（苛判漂移：学生已表达答案要点等价含义，模型自行提高
完整度门槛）；5× miss→partial + 1× miss→hit（宽判失守：给错误作答部分/全部学分）；
两方向同时失守说明不是阈值问题，是**裁量本身不稳定**。

## 裁决（按 §3 预先约定的分支）

**记录轨（确定性判分）为权威。** agent 评语照常显示（教练的话语），UI 披露
「量表记录以判分引擎为准」；divergence 事件供教师复核。产品形态不变，仅披露口径
落入预定分支。附带收益：本实验本身即答辩素材——「为什么判分零 AI 裁量」从设计
哲学变成了同语料对照实验的实证结论（36.5pt 差距 + 全错判掌握违规）。

限制声明：本实验 provider 为 claude-agent（不响应 temperature、无 schema 强制），
正式 provider 下直接评判的一致率会改善，但宏命中率差距与双向裁量漂移属模型行为
而非采样噪声，结论预期稳健。语料为单题（膜题）52 例，扩语料后可复验。
