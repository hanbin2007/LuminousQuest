## 一、系统性问题（影响多题）

### A. 六道定量计算小题统一标了 E3+P4，与两节点定义不符

涉及：1-3、3-4、4-4、5-3、6-3、8-3（全部为电子转移计量计算）。

- E3 的量表定义是「能量转化并非 100%，部分以热等形式损耗；燃料电池效率通常高于火力发电」——与计量计算无关；
- P4 的定义是电子/离子**转移方向**——计算题只用到方向的前置理解，考的不是方向本身；
- 这类题的模型归属应是 **P6**（「两极电子转移数相等」即电子守恒条款）。

建议：六题主指标改 P6（3-4 可保留 D3，6-3 可加 P7，见下）。
需要确认：老师是否把 E3 当作「定量计算能力」在用？若是，说明 E3 的量表陈述与老师的心智模型有歧义——按 Core 文档的原则，这种情况应修订量表措辞而不是迁就标注，否则抽取和判分会系统性错位。

### B. P7（总反应式）从未被使用，但三道题正是 P7 的定义任务

6-1(3)、7-3(1)、8-4(1) 都要求「半反应相加消项得总反应」，这就是 P7 的 statement 原文。建议三处补标 P7。

### C. 第 7、8 题的参考答案与知识模型 P1、E2 的现行陈述正面冲突（最重要的一条）

7-3(2) 与 8-4(2) 的参考答案是：「原电池不一定是氧化还原，两极存在电势差也可」（浓差/络合驱动电池，总反应 Cu²⁺ + 4NH₃ = [Cu(NH₃)₄]²⁺ 无化合价变化）。但现行量表：

- **P1**：「前提是一个**自发的氧化还原反应**……」
- **E2**：「电能来源于**自发氧化还原反应**释放的化学能」

按现行量表判分，学生答出教师期望的正确答案反而无法命中任何节点，甚至可能被 contradiction 规则误伤。三个可选处置，需教师定：

1. 修订 P1/E2 陈述，加限定（如「中学模型内通常为……；浓差/络合电池为拓展边界」），并考虑是否新增边界节点；
2. 这两小题按 `beyondSyllabus` 策略处理（超纲但正确 → hit，不入 P1/E2）；
3. 这两小题不进判分，仅作教学讨论题。

同时注意 8-1(3) 关键词说「S 到 Cu₂S 化合价降低、得电子」（半反应有化合价变化）而 8-4(2) 说总反应无化合价变化——两处并不矛盾（半反应有、总反应无），但抽取 prompt 和量表若不区分「电极反应层」与「总反应层」，容易误判学生的表述为自相矛盾。

### D. 7-1(3) 的判断依据（浓差电势）在模型中没有落点

「两侧铜离子浓度差异 → 电极电势不同 → 存在电势差」这一判断路径，15 个节点均不覆盖（P2 是氧化性/还原性路径）。7-1 现标 D1、D4、P2、P6，其中 P2 对 (3) 小题不成立。与 C 同属「浓差/络合电池超出现行模型」的覆盖缺口，建议合并裁决。

## 四b、评分关键词的化学疑点（双路评审新发现，需老师确认）

1. **【高】8-2(2) 离子迁移方向疑误**：参考答案写「S²⁻ 从右室移向左室」。按两极反应做电荷平衡——左室（正极）消耗 Cu²⁺、右室（负极）生成 [Cu(NH₃)₄]²⁺——为维持电中性，应是阴离子（电解液中的 SO₄²⁻）**从左室移向右室（负极）**，与量表 P4「阴离子移向失电子场所」一致；「S²⁻ 从右向左」方向相反，且 S 在本装置中以固体形式存在于电极，游离 S²⁻ 载流的说法也存疑。此关键词若不修正，将与 P4 判分互相矛盾，不能进入判分配置。（2026-07-22：已对照原文档截图复核，转录无误，疑点属参考答案本身；验算过程已交张涵彬转陆老师确认。**同日已解决**：陆老师提供的解析版 answer-key.md 删去了「S²⁻ 从右室移向左室」，改为阳膜下 [Cu(NH₃)₄]²⁺ 迁移破坏氨气再生循环的功能论证，疑点消除，本条关闭。）
2. **【中】7-3(2)、8-4(2) 参考表述建议精化**：「没有氧化还原反应／原电池不一定是氧化还原」易造成新误解——两极的半反应仍是氧化、还原（有化合价变化），只是**总反应无化合价净变化**。建议参考要点改为：「合并后的总反应无元素化合价净变化，但负、正极仍分别发生氧化、还原半反应；电池由电势差驱动、净过程自发」。转 eval golden set 与判分配置时按此修正表述，不照录原关键词。（2026-07-22 解析版中老师保留了原表述——参考答案以老师版为准；判分层依 rubrics v1.2 的 p1-hit 分支区分「总反应无净变化」与「两极仍有氧化还原」，学生答出层次区分时同样判 hit，不因表述差异扣分。）
