## 第 1 题　K—O₂ 电池

- 1-1：负、正
- 1-2：a、b
- 1-3：1:1
- 1-4：不能、防止 K 与 O₂ 直接反应

解析：由图可知，b 极氧气转化为超氧化钾，氧的化合价降低，b 电极为正极；a 电极为原电池的负极，单质钾片失去电子发生氧化反应生成钾离子，电极反应式为 K − e⁻ = K⁺。

## 第 2 题　Zn-MnO₂ 双隔膜电池

- 2-1：负、正
- 2-2：Zn − 2e⁻ + 4OH⁻ = Zn(OH)₄²⁻
- 2-3：Ⅰ
- 2-4：化学能、电能

解析：根据图示的电池结构和题目所给信息（KOH 溶液中，Zn²⁺ 以 Zn(OH)₄²⁻ 存在）可知，Ⅲ区 Zn 为电池的负极，电极反应为 Zn − 2e⁻ + 4OH⁻ = Zn(OH)₄²⁻；Ⅰ区 MnO₂ 为电池的正极，电极反应为 MnO₂ + 2e⁻ + 4H⁺ = Mn²⁺ + 2H₂O。

## 第 3 题　微生物脱盐电池

- 3-1：负、正
- 3-2：CH₃COO⁻ + 2H₂O − 8e⁻ = 2CO₂↑ + 7H⁺
- 3-3：（1）阴、阳；（2）要实现海水淡化，Cl⁻ 移向负极（a 极）、Na⁺ 移向正极（b 极）
- 3-4：58.5

解析：据图可知 a 极上 CH₃COO⁻ 转化为 CO₂ 和 H⁺，C 元素被氧化，所以 a 极为该原电池的负极，则 b 极为正极。

## 第 4 题　血糖微型电池

- 4-1：正、负
- 4-2：O₂ + 2H₂O + 4e⁻ = 4OH⁻
- 4-3：（1）CuO；（2）Cu₂O；（3）氧化铜将葡萄糖氧化为葡萄糖酸，自身被还原为氧化亚铜，氧化亚铜在 b 电极失电子又生成氧化铜。催化作用
- 4-4：0.2

解析：由装置图中信息可知，a 电极为正极，发生反应 O₂ + 2H₂O + 4e⁻ = 4OH⁻；b 电极为负极，负极的电极材料为纳米 CuO/导电聚合物（**注意：负极的电极材料并不等同于负极的电极反应物**），b 电极的整个过程为：氧化铜将葡萄糖氧化为葡萄糖酸，自身被还原为氧化亚铜，氧化亚铜在 b 电极失电子又生成氧化铜。

## 第 5 题　钠-氯储能电池

- 5-1：Na₃Ti₂(PO₄)₃ − 2e⁻ = NaTi₂(PO₄)₃ + 2Na⁺
- 5-2：NaCl 溶液
- 5-3：释放、35.5
- 5-4：NaCl 溶液和 CCl₄ 不互溶，Cl₂ 易溶于 CCl₄、难溶于 NaCl 溶液，使得 Cl₂ 无法与 Na₃Ti₂(PO₄)₃ 接触。

解析：观察题干中的俯视图，Cl₂ 具有强氧化性，容易得电子，Cl₂ 所处的多孔活性炭电极为正极（**注意，这里的"正极"包含电极材料和电极反应物，电极材料是多孔活性炭，电极反应物是氯气**），正极反应：Cl₂ + 2e⁻ = 2Cl⁻；负极为钛电极，负极反应：Na₃Ti₂(PO₄)₃ − 2e⁻ = NaTi₂(PO₄)₃ + 2Na⁺。此电池的特点是不需要离子交换膜就可以将两极反应物隔开，两极之间充斥着 NaCl 溶液，NaCl 溶液和 CCl₄ 不互溶，Cl₂ 易溶于 CCl₄、难溶于 NaCl 溶液，使得 Cl 无法与 Na₃Ti₂(PO₄)₃ 接触。

## 第 6 题　全铁液流电池

- 6-1（1）：负、Fe − 2e⁻ = Fe²⁺
- 6-1（2）：正、2Fe³⁺ + 2e⁻ = 2Fe²⁺
- 6-1（3）：Fe + 2Fe³⁺ = 3Fe²⁺
- 6-2（1）：阴
- 6-2（2）：若为阳膜，电池不工作时 Fe³⁺ 会扩散向负极与 Fe 反应
- 6-3：1.5

解析：本题的关键在于根据题目信息和装置图信息分析出两极。题干中描述有一极为负载铁的石墨电极，那么电极 b 不可能是铁电极，因为电极 b 周围有 Fe³⁺，Fe³⁺ 与 Fe 不能共存，因此电极 a 为铁电极，那么 a 为负极、b 为正极，电极反应也比较清楚了。

## 第 7 题　热再生氨电池

- 7-1（1）：负、Cu − 2e⁻ + 4NH₃ = [Cu(NH₃)₄]²⁺
- 7-1（2）：正、Cu²⁺ + 2e⁻ = Cu
- 7-1（3）：甲室 Cu 通入 NH₃ 后铜离子浓度减小，两侧铜离子浓度存在差异，电极电势不同，存在电势差
- 7-2（1）：阴
- 7-2（2）：若为阳离子交换膜，[Cu(NH₃)₄]²⁺ 会迁移至乙室，甲室无法低温热解实现硫酸铜溶液再生
- 7-3（1）：Cu²⁺ + 4NH₃ = [Cu(NH₃)₄]²⁺
- 7-3（2）：不同意，即便没有氧化还原反应，但是两极存在电势差也能形成原电池

解析：根据题目信息和装置图信息，"低温热解"是 [Cu(NH₃)₄]²⁺ 受热分解为铜离子和氨气，甲室中要生成 [Cu(NH₃)₄]²⁺，电极反应总是 Cu − 2e⁻ + 4NH₃ = [Cu(NH₃)₄]²⁺（或者也可以理解为一开始甲室乙室的铜离子浓度相等，但是甲室中通入氨气后，铜离子浓度减小，那么甲室铜就会失电子）。

## 第 8 题　S/Cu₂S 转化电池

- 8-1（1）：正、S + 2Cu²⁺ + 4e⁻ = Cu₂S
- 8-1（2）：负、Cu₂S − 4e⁻ + 8NH₃ = 2[Cu(NH₃)₄]²⁺ + S
- 8-1（3）：S 和 Cu₂S 之间相互转化，S 到 Cu₂S 化合价降低，得电子，做正极
- 8-2（1）：阴
- 8-2（2）：**若为阳膜，电极 b 生成的四氨合铜离子会迁移到 a 极，那么不符合题干所述"将一侧电解液加热，产生的气体（氨气）通入另一侧，可使电池重复工作"**
- 8-3：1:2
- 8-4（1）：Cu²⁺ + 4NH₃ = [Cu(NH₃)₄]²⁺
- 8-4（2）：无化合价变化，原电池不一定是氧化还原，两极存在电势差也可

解析：根据题干"S 和 Cu₂S 的相互转化"，结合化合价变化可得电极反应：左室电极 a 为正极，电极反应 S + 2Cu²⁺ + 4e⁻ = Cu₂S；右室电极 b 为负极，电极反应 Cu₂S − 4e⁻ + 8NH₃ = 2[Cu(NH₃)₄]²⁺ + S，加热时 [Cu(NH₃)₄]²⁺ 释放 NH₃，据此解答。
