## 误区诊断新增点

- **7-3(2)、8-4(2)**：学生若「同意原电池一定有化合价变化」→ 命中 **P1-M3**；答出「电势差也可构成原电池」→ P1 hit（v1.2 拓展条款）。这两小题从「与量表冲突」转为 P1-M3 的专用诊断题。
- **P1-M3 的判定链路（重要限制）**：抽取链路是闭集事实槽转录，误区 id 在 schema 中只是不带语义的枚举——误区的实际判定依靠「事实槽值 → 规则层映射」。因此 P1-M3 的自动判定需在题目转 config 时为 P1 设计专用事实槽（如「是否认为总反应必须有化合价变化」），由规则层映射到 P1-M3；rubrics v1.2 的 p1-miss 描述已为此预留判分口径。在该槽位与 eval 语料落地之前，7-3(2)、8-4(2) 一律走「待教师复核」，不进自动判分。
- **8-1(3) 与 8-4(2) 的关系**：半反应层有化合价变化、总反应层没有——抽取 prompt 与判分需区分层次，不得将学生同时陈述两者判为自相矛盾（已在量表 P1 陈述中写明「两极仍各自发生得失电子的半反应」）。
