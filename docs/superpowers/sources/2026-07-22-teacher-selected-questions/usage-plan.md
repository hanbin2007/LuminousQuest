# 陆老师选题使用方案

日期：2026-07-22
输入：8 道原电池高考题（[`questions.md`](questions.md)）+ 定稿标注（[`final-annotations.md`](final-annotations.md)）
约束：功能冻结 2026-08-08；8/12 发布候选；不新增大模块。

## 定位判断

这批题是**判分型真题**（填空/简答 + 节点映射 + 评分关键词），与 `pretest.json` 的题目 schema 同构，与训练案例的「三维度空白框分析」格式**不**同构。因此它们的自然落点是「测评链路」（前测/后测/eval），不是「训练案例」。

## 三层使用方案

### T0 立即可用（零代码，本周）

1. **eval golden set 语料**：每小题的评分关键词就是期望抽取结果（参考答案一律以 answer-key.md 解析版为准；8-2(2) 疑点已由解析版解决）。为每小题编写正确作答、典型错误作答（含 P1-M3、P4-M3 等误区）与边界作答，转入 `eval/cases/`——这直接充实 M1c 尚欠的语料量（150/5/3 目标）。Q7/Q8 专供拓展情境与 P1-M3 对抗样本（兼作 M5 延期项 D1 的否定句对抗语料）。
2. **答辩证据**：真题→节点映射表是「量表对接高考评分惯例」的直接证据，编入教师决策日志与答辩包（评委问「标准从哪来」时，这是比课标更具体的一环）。

### T1 前测「高考真题诊断」段（已实现，2026-07-22）

落地形态调整：经真实学生数据（20 份纸面作答含 Q1）评估后，Q1 改为进入**前测**而非冷迁移后测——同一批学生的纸面作答可直接校准工具判分链路，且补齐前测对 P1 的证据缺口。已实现（规格 `2026-07-22-pretest-exam-q1.md`，pretest.v1.2）：

- Q1 拆 4 小题：1-1/1-2/1-3 为确定性选择题（错误选项挂误区标签），1-4 简答走 AI 抽取（参考答案含真实作答中的另类正确理由）。
- question 新增可选 `group` 字段（共享题干+装置图），「高考真题」徽标区块；`assets/exam/` 存教师原题图（manifest 豁免机检并注明理由）。
- demo 冻结口径：预置「已答未判分」，演示画像与雷达/3D 完全不变。
- 备选 Q6（全铁液流）留二期题库。

### T2 二期（真题题库模块）

8 题全量入库，按画像 `weakNodeIds` 定向推荐（衔接「自适应选题」路线：真实作答数据回流后校准）。分层：

| 题 | 层级 | 说明 |
|---|---|---|
| Q1 K—O₂ | 基础全面 | 后测/入门检验 |
| Q2 Zn-MnO₂、Q3 微生物脱盐 | 中阶 | 膜判断、介质匹配（P3 重点） |
| Q4 血糖电池 | 中阶·D5 专项 | 4-3 是 D5「场所/反应物两层」的最佳诊断题 |
| Q5 钠-氯、Q6 全铁液流 | 中高阶 | 计量与总反应（P6/P7） |
| Q7 热再生氨、Q8 S/Cu₂S | 拓展层 | P1-M3 边界诊断；真实作答校准前不进自动判分，先走「待教师复核」 |
