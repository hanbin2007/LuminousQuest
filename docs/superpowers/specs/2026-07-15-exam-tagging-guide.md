# 高考真题筛选与标注指南(v1)

日期:2026-07-15
用途:近五年高考电化学真题 → 前测题改编 / 训练与冷迁移案例情境 / 新误区候选 / 评分惯例证据。
节点与误区 id 以量表 v1 为准(2026-07-14-rubric-draft-v0.md);标注可先用表格(飞书/Excel/CSV 均可),后转 JSON。

## 三层漏斗

1. **范围粗筛**:仅原电池入库;纯电解池归档标"二期";混合题拆出原电池部分。
2. **情境归类与去重**:按电极类型(全活性/半重合/全惰性)、介质(酸/碱/盐/熔融)、结构(单液/盐桥/隔膜/特殊)归类,同类型留 1~2 个代表;记录我们案例库(锌铜/铝-空气/氢氧燃料/甲烷)未覆盖的情境作为二期与冷迁移候选。
3. **选项级精标**:仅对入选 30~50 题;流程=codex 批量预标注 → 人工复核。

## 题目层字段

| 字段 | 说明 | 示例 |
|---|---|---|
| exam_id | 年份-卷别-题号 | 2024-新课标I-12 |
| battery | 电池情境 | 锌-空气电池 |

| medium | 酸 / 碱 / 盐 / 熔融 | 碱 |
| structure | 单液 / 盐桥 / 隔膜 / 特殊 | 隔膜 |
| q_type | 选择 / 填空 / 综合 | 选择 |
| nodes | 涉及节点 id(多选) | D5,P4,P6 |
| usage | 前测改编 / 案例参考 / 冷迁移候选 / 仅归档 | 前测改编 |
| figure | 图是否需按 STYLE.md 重绘 | 需重绘 |
| notes | 备注(评分标准片段等) | 官方答案体现跟随性给分 |

## 选项层字段(每选项一行)

| 字段 | 说明 | 示例 |
|---|---|---|
| option | 选项号 | B |
| verdict | correct / wrong | wrong |
| nodes | 该选项考察的节点 id | P4 |
| error_id | 错误选项命中的误区(量表内) | P4-e2(电子经盐桥) |
| new_error | 量表中无对应误区时,一句话描述 → 进"新误区候选清单" | 认为盐桥中电子与离子同时移动 |

## 分流规则

- 错误选项精准命中单一误区、干扰性强 → **前测改编池**(误区探测器)。
- 情境新颖、信息完整(有装置图/介质/反应) → **案例/冷迁移候选**。
- 官方评分标准体现阅卷惯例(跟随给分、介质扣分口径) → 摘录进 notes,作为裁量 §2/§8 的证据。
- 新误区候选每周汇总一次,人工审核后增补进量表(惰性增厚流程)。
