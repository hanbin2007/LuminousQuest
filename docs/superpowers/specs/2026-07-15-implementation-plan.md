# 实现计划 v1

日期:2026-07-15
前置:设计规格与量表 v1 均已审定(陆老师认可);比赛时间 2026-09,目标 8 月中旬交陆老师试用版,9 月前完成彩排。
分工铁律:网页视觉/3D/风格规范 = Fable 5 亲自;其余实现+审查 = gpt-5.6-sol(effort 一律 `max`);每里程碑评审 opus-4.8 + gpt-5.6-sol 双路;里程碑终审 gpt-5.6-sol(max)+ Fable 5。

## M0 工程骨架(codex)

- 单仓库:Vite + React + TS 前端,`server/`(Hono)后端;vitest;pnpm。
- Node 服务三职责:静态托管;`/api/llm` 代理(key 在 `.env`,自动录制/回放,演示模式按脚本步骤 ID 取回放、开发模式按归一化 prompt 哈希缓存);`/api/config` 热加载。
- 启动:检测空闲端口,占用则换(严禁静默抢占);`start.command`/`start.bat` 双击脚本。
- `LLMProvider` 抽象(chat / vision / structured-output),DeepSeek、通义、智谱三个 adapter 骨架(择主 provider 待 M1 eval)。
- 全部 `config/` 文件的 zod schema + 启动校验(报文件/字段/原因)。

产出验收:双击启动、空配置跑通、schema 校验报错友好、录制回放单测通过。

## M1 判分核心 + 配置内容(codex;本项目技术心脏)

- **教材交叉核验**(首个任务):codex 对量表 v1 逐节点核对课标/教材表述,产出核验报告,疑点回报人工裁决。
- 量表 v1 转录为 `knowledge-model.json`(15 节点,3D 坐标继承陆老师模型图布局,含 D5 跨轴边)+ `rubrics.json`(依赖关系、跟随性错误锚点、权重)。
- `pretest.json`(功能四要素判分+抽象度检查+干扰项组件集+3 道题);`cases/`4 个(锌铜/铝-空气/氢氧燃料/甲烷冷迁移,含三级脚手架引导问题、四连问、参考要点);`scaffold-policy.json`(默认策略,标注"待教师细调");`prompts/`。
- 规则引擎:搭建器拓扑判分(图规则:四要素、连通、环、方向一致性)。
- 化学式判分器:tokenizer + 三守恒校验 + 按案例等价集比对;符号条输入组件的归一化。
- 抽取管线:闭集抽取(node/error id)+ 引用原文校验(归一化模糊匹配,错别字容忍)+ schema 重试 + 降级"待教师复核"。
- 画像与判分聚合(跟随性错误按裁量 §2 默认)、脚手架确定性状态机、苏格拉底微循环(轮数/动作白名单/泄题检测/回退)。
- **eval harness**:golden set v0(张涵彬手写学生腔样本 + 合成边界样本 + 每误区至少 1 条)、蜕变测试(改写不变分、换名不变分、加废话不降级)、一致性(每 case×3)、"全错判掌握"单独盯(≤2%);provider 选型跑分(含真实手绘样本测视觉)。

产出验收:eval 达标(节点判定 ≥90%,严重误判 ≤2%);确定性逻辑单测全覆盖。

## M2 前测模块 + 诊断呈现(UI 实现 codex,视觉 Fable 5)

- **Fable 5 先行**:设计 token(校色系统:校深蓝 #2B5CA8=装置、校青 #2CB8B8=原理、琥珀 #E8960C=能量;纸白 #F7F8F5/墨色 #1F262B;思源宋体 Heavy 子集+系统黑体栈)+ `assets/STYLE.md`(教辅彩色插图规范、出图 prompt 模板与参考图)。
- codex 按 STYLE.md 批量生成透明底 PNG 组件素材(2x),Fable 5 逐批终审一致性。
- 搭建器(功能化组件+干扰项、吸附连接、拓扑实时校验)、3 道题、手绘彩蛋(canvas→vision)、雷达图(ECharts,dataviz 校验对比度)、证据清单、电子流进度母题、红笔批注反馈样式(Fable 5 出样式规范)。

## M3 训练模块(codex)

- 案例流:素材递进(简图→剖面)、三级脚手架 UI(按 P2→P3→P4→P5 阶梯引导+四连问)、即时反馈(答对什么/错在哪/下一步想什么)、苏格拉底微循环接入、冷迁移后测(强制三级、无提示)、前后雷达对比。

## M4 外显与交付

- **3D 思维模型(Fable 5 亲自)**:三轴网络=陆老师模型图三维数字化;hit 全亮/partial 半亮/miss 暗;前测初始化点亮;D5 跨轴边;emissive+精灵光晕(不开全屏后处理);关灯时刻转场;核显笔记本实测帧率。
- 教师视图+班级汇总(批量导入→班级雷达+误区 Top N)(codex)。
- 演示模式(预置会话+一键切回放)、单文件打包(bun compile,Win/mac)、比赛运行手册(2min 稿/5min 脚本/3min 答辩包)、断网 10 分钟彩排 ≥3 次。
- 安全练习:手绘藏指令、嘴硬学生、套量表三项红队用例进 eval。
