# M1a 教材交叉核验

日期：2026-07-15  
核验对象：`knowledge-model.v1` 的 15 个实现节点  
结论：14 个权威量表节点均可按高中通行口径落地；4 个措辞或边界问题与 1 个节点数量问题已列为疑点，但均按默认值继续，不阻塞 M1a。

## 核验依据

1. [人教版《普通高中教科书·化学 选择性必修1 化学反应原理》](https://r1-ndr.ykt.cbern.com.cn/edu_product/esp/assets_document/3502fe81-b23e-4f68-aa3d-7921e7932ec9.pkg/pdf.pdf)，重点核对第四章章首及第一节“原电池”（教材页 95-102）。
2. [《普通高中化学课程标准（2017 年版 2020 年修订）》](https://www.pep.com.cn/xw/zt/rjwy/gzkb2020/202205/P020220517517668288401.pdf)，重点核对“化学反应与能量”内容要求及“能分析、解释原电池和电解池工作原理”的学业要求。
3. 量表 v1 是本项目权威内容源；教材核验只检查是否符合通行教学表述，不在本阶段擅自覆盖已冻结内容。

教材关键口径：第 95 页明确区分电子导体与离子导体，并说明化学能与电能的直接转化须通过氧化还原反应；第 97 页以锌铜原电池说明负极氧化、正极还原、电子与盐桥离子运动、电极反应式和守恒；第 101 页以酸性氢氧燃料电池说明惰性 Pt 场所、外部反应物与介质影响，并给出燃料电池和火力发电的转化率比较。

## 逐节点结果

| 节点 | 教材/通行表述核对 | 结论与实现处理 |
|---|---|---|
| D1 失电子场所 | 教材第 97 页把 Zn 极标为负极并在其上发生氧化反应；第 101 页氢气在 Pt 负极界面失电子，说明场所不必等于固体反应物 | 一致；保留“活性材料或惰性电极”的统一模型表述 |
| D2 电子导体 | 第 95 页使用“电子导体”，第 97 页明确电子由负极通过导线流向正极 | 一致 |
| D3 离子导体 | 第 95 页使用“离子导体”；第 96-97 页说明盐桥离子可自由移动、移除盐桥后电流归零，并给出 K⁺/Cl⁻ 迁移 | 一致；盐桥作为离子导体实现，不单列必备节点 |
| D4 得电子场所 | 第 97 页 Cu²⁺ 在铜片上得电子；第 101 页 O₂ 在 Pt 正极界面得电子 | “得电子场所”一致；“材料通常不参与、常为惰性”不是所有电池的普遍必要条件，暂保留量表原文并列疑点 Q2 |
| D5 场所与反应物之辨 | 锌铜案例中 Cu 是场所、Cu²⁺ 是反应物；燃料电池中 Pt 是场所、H₂/O₂ 是外部通入反应物 | 内容一致；“双重含义”是项目模型术语，不是教材原句，但有明确教材事实支撑 |
| P1 反应前提 | 第 95 页说明直接转化以氧化还原反应为基础；第 97 页说明其他自发氧化还原反应可设计成原电池 | 一致；“两个半反应分隔在两个场所”符合第 97 页半电池表述 |
| P2 电极反应物 | 第 97 页 Zn 失电子、Cu²⁺ 得电子；第 101 页 H₂ 为燃料、O₂ 为氧化剂 | 一致 |
| P3 电极产物 | 第 97 页给出 Zn²⁺/Cu；第 101 页明确氢氧燃料电池可用酸性或碱性电解质，并按酸性介质写 H⁺/H₂O | 一致；碱性铝-空气产物表示不在该节直接给出，列疑点 Q3 |
| P4 过程 | 第 97 页给出电子由负极经导线到正极、Cl⁻ 到负极侧、K⁺ 到正极侧 | 锌铜与高中通用模型一致；选择性膜案例应按实际载流离子解释，列边界疑点 Q4 |
| P5 现象 | 第 96 页实验以电流表偏转为证据；第 97 页反应式可推出 Zn 溶解、Cu 沉积，教材任务要求标明电子与阴阳离子方向 | 一致；外电路常规电流方向与电子相反是通行物理口径 |
| P6 电极反应式 | 第 97 页“方法导引”明确质量守恒、离子电荷守恒和电子转移守恒，并给出两极及总反应式 | 一致；实现中把“质量守恒”落实为逐元素原子数守恒 |
| P7 总反应整合（暂定） | 教材第 97 页由两个电极反应得到总反应，且总反应不含电子 | 教学内容成立，但量表 v1 表内没有 P7；这是为满足“15 节点”契约从 P6 拆出的暂定节点，列最高优先级疑点 Q1 |
| E1 释能方向 | 第 95 页明确原电池将化学能转化为电能，电解池方向相反，且称为直接转化 | 一致 |
| E2 能量来源 | 第 95、97 页把电能输出与氧化还原反应及反应物氧化还原能力关联 | 一致 |
| E3 转化限度 | 教材第 101 页称燃料电池转化率超过 80%，高于仅 30% 多的火力发电；通行口径也承认有热等损耗 | 核心意思一致；量表写“直接燃烧发电”，教材写“火力发电”，比较对象需收窄，列疑点 Q5 |

## 疑点清单

| id | 疑点 | 当前默认 | 人工裁决期限 |
|---|---|---|---|
| Q1 | 量表标题及计划写“15 节点”，但 v1 表内只有 D1-D5、P1-P6、E1-E3，共 14 个 | 增加次要节点 P7“总反应整合”，权重 1，依赖 P6；不改既有 14 节点内容 | 2026-07-17 23:59:59 +08:00 |
| Q2 | D4 的“材料通常不参与反应，常为惰性电极”容易被读成所有正极的必要条件 | 按权威量表原文转录；反馈与案例强调“场所材料不等于被还原微粒”，不把“惰性”作为通用判分门槛 | 同上 |
| Q3 | 碱性铝-空气负极产物可按课程/条件写 `AlO2^-` 或 `[Al(OH)4]^-`，实际还可能涉及 Al(OH)₃ 与钝化 | 案例等价集接受两条守恒的碱性表示；不接受最终式残留游离 H⁺ | 同上 |
| Q4 | “阳离子到正极、阴离子到负极”适合高中溶液/盐桥模型，但质子交换膜等装置只允许特定离子通过 | 量表判分沿用高中通行方向；具体案例优先核对配置的实际载流离子 | 同上 |
| Q5 | E3 的效率比较对象在量表与教材间分别为“直接燃烧发电”和“火力发电” | 保留权威量表原文；案例反馈采用教材更窄的“火力发电”口径 | 同上 |
