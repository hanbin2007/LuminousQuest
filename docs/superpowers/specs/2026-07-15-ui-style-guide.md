# UI 风格指南(视觉施工图)

制定:Fable 5(视觉负责人)| v1,2026-07-15
效力:M2/M3 全部 UI 实现必须遵循;色值/字号/动效一律取 `src/styles/tokens.css` 的 token,组件内禁止硬编码。视觉验收由 Fable 5 终审。

## 1. 整体气质:「新教辅 × 关灯舞台」

- 全局亮态:冷调纸白 + 墨色文字,像一本现代教辅。**唯一的强色是三维度色**(校深蓝=装置/校青=原理/琥珀=能量),它们在雷达图、脚手架框架、节点点亮、3D 三轴中始终一一对应——颜色即语义,任何组件不得挪用三维度色做装饰。
- 暗态只存在于模块三的 3D 舞台(`.stage-dark` 作用域),入场用"关灯时刻"转场(`--dur-stage`,全项目唯一的大动效)。
- 术语与文案一律采用陆老师模型图原词(失电子场所、电子导体、离子导体、得电子场所、电极反应物、电极产物…)。按钮动词直述结果("提交作答""进入训练"),全程一致。

## 2. 三个签名元素(实现规格)

**① 电子流进度条(母题)**
- 顶部流程指示(前测→训练→外显)是一条 `--eflow-track` 轨道,已完成段用 `--eflow` 填充,**流动光点**(2~3 个小圆,CSS animation 沿轨道匀速)只在"刚完成一步"后运行 2 秒然后停驻——常驻流动会喧宾夺主。
- 答对反馈可复用:该题卡片边缘一次性流过一段电子流光点(150ms 延迟后 600ms 完成)。
- reduced-motion 下退化为纯色填充,无光点。

**② 红笔批注(AI 反馈的形态)**
- AI 反馈不用聊天气泡。呈现为**页边批注卡**:左侧 3px 竖线 + 手写感倾斜 -1deg 的卡片,错因用 `--annotate-red`(线+标题),启发式追问用 `--annotate-blue`。
- 批注卡内部结构固定三行:答对了什么(墨色)/ 错在哪里(红)/ 下一步想什么(蓝)。
- 批注出现动效:从对应作答区域侧边 8px 滑入 + 淡入(`--dur-base`),像老师落笔。

**③ 关灯时刻(进入模块三)**
- 触发:学生完成案例最后一个知识点。序列(总时长 `--dur-stage`):页面亮度渐降(overlay 到 75% 暗)→ 3D 画布从中心淡入 → 三轴骨架先亮 → 已掌握节点按"前测→训练"时间顺序逐个点亮(每个间隔 80ms)。
- 这是演示脚本的高潮拍,实现时给一个 `?stage-demo` 参数可直接重放整个序列。

## 3. 布局与组件规范

- **页面骨架**:内容最宽 `--content-max`,训练页左右分栏(素材区 5 : 作答区 7),分栏间距 `--space-6`。
- **卡片**:`--paper-raised` + `--radius-lg` + `--shadow-card`;卡片内标题用 `--font-display` 的 `--text-xl`,正文 `--font-body`。
- **标题层级**:模块页大标题=display 字体 `--text-display`;其余标题克制,最多三级。
- **搭建器画布**:`--paper-sunken` 底 + 极淡方格纸纹理(1px `--hairline` 网格,24px 间距,透明度 40%)——实验记录本的隐喻;组件吸附成功时目标位置闪一次 `--dim-principle-soft` 底光。
- **雷达图**:三轴用三维度色;前测/后测对比时,前测为 40% 透明度同色填充,后测为实线。轴标签用模型图原词。图表实现前对照 dataviz 校验对比度。
- **方程式输入**:符号条(e⁻、上下标、⇌、→、常用离子)置于输入框上沿,等宽字体显示;判分证据引用时用 `--paper-sunken` 底色内联高亮学生原文。
- **状态语义**:hit=绿(仅边框/图标,不大面积填充)、partial=琥珀、miss=批注红、unassessed=灰。**灰和红永远不得混用**——"没测到"不是"错"。
- **按钮**:主按钮墨色底白字(`--radius-md`),次按钮描边墨色;禁用态用 `--ink-faint`。焦点环 `--focus-ring` 全组件必备。

## 4. 教师视图与班级汇总

- 数据密度高于学生页,允许表格;但证据链呈现沿用批注卡语言(教师看到的和学生看到的是同一套视觉,建立信任)。
- 班级雷达:均值实线 + 分布带(同色 15% 透明度);节点错误率排行用横向条,条色=所属维度色。

## 5. 禁止项

玻璃拟态、霓虹渐变、大面积阴影、彩色大标题、装饰性 emoji、聊天气泡式 AI 形象、常驻循环动画、三维度色之外的第四种强色。任何页面同屏强色不超过三维度色 + 一个批注色。

## 6. 验收方式

M2/M3 每个页面完成后截图(亮态全流程 + 关灯序列帧),由 Fable 5 对照本指南逐条签核;不符退回。字体子集覆盖:量表/案例/界面全字表 + 系统栈兜底(教师改 JSON 引入新字不缺字)。
